📊 What is the Game Sheet?
Controls every mission players see on their My XP dashboard. Add a row = new mission on the site. No coding needed.

📍 How to Access

  • Open the Vote Reeder Events Google Sheets workbook
  • Click the Game tab at the bottom
  • Row 1 is the header — never delete or edit it
  • All missions start on Row 2

📋 Column Reference

ColumnWhat It DoesRules
A — CategoryGroups missions on the dashboardRequired. Use dropdown only
B — ActionMission name players seeRequired. Keep short and clear
C — XPPoints this action is worthRequired. Numbers only
D — VerificationHow the action gets verifiedRequired. Auto / Event Code / Honor System / Self-report / Screenshot / ActBlue Order # / Username / Quiz
E — ActiveShows or hides the missionRequired. YES = visible, NO = hidden
F — IconEmoji shown on mission cardAny emoji
G — DescriptionShort description for playersKeep under 80 characters
H — End DateWhen mission expiresFormat: MM/DD/YYYY
I — RepeatableCan be done more than onceYES or NO

📦 Valid Categories

🎓 Learning📅 Events🚪 Field Work🗳️ Voter Registration📱 Social Media🤝 Recruitment💰 Donation📋 Volunteer⭐ Bonus🎉 Fun

✅ How to Add a New Mission

  • Scroll to the first empty row below all existing missions
  • Select a Category from the dropdown in column A
  • Fill in Action (B), XP (C), Verification (D)
  • Type YES in column E (Active) and column I (Repeatable)
  • Mission appears on the site within minutes
✓ To hide a mission: change column E from YES to NO.
🏆 What is the Rank Sheet?
Controls the 6 player ranks — names, XP thresholds, rewards. The site reads this automatically.

📝 Current Rank Structure

RankXP RequiredReward
🌱 Supporter0 XPCampaign badge + welcome pack
📋 Organizer300 XPGuild access + mission unlocks
📣 Advocate1,000 XPLeaderboard featured + certificate
🏗️ Builder3,500 XPVIP event invites + name on site
⚗️ Catalyst8,000 XPGuild leadership + campaign role
🏆 Change Champion15,000 XPHall of Fame + Campaign Legacy

✏️ How to Adjust XP Thresholds

  • Find the rank row, click XP Required (column B), type new number, press Enter
  • Keep ranks in ascending order
  • Site updates automatically
💡 If too few people reach Builder, lower it from 3,500 to 2,500 to keep players engaged.

⚠️ What NOT to Change

  • Do NOT delete any rank rows — the site expects exactly 6 ranks
  • Do NOT change rank names without also updating the site code
⚠️ If you accidentally break the sheet, contact your developer to restore from backup.

🚀 When do I need to deploy?
Google Sheet changes update the site automatically. You only deploy when HTML, CSS, or JS files change.

📋 Deploy vs No Deploy

ChangeDeploy Needed?
Add/edit event, mission, video, rank in Google Sheets✅ No — auto
Add/remove admin users in Admin sheet✅ No — auto
Edit any .html, .css, or .js file⚠️ Yes — deploy
Add new images to the images folder⚠️ Yes — deploy

🚀 How to Deploy to Netlify

  • Open your site folder, make changes in Notepad++, save (Ctrl+S)
  • Go to app.netlify.com → find site comforting-chebakia-67f8df
  • Drag the entire site folder onto the deploy area
  • Wait ~30 seconds → visit votereederxp.com to confirm
⚠️ Always drag the WHOLE folder — not just the changed file
