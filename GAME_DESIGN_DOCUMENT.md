# VoteReederXP — Game Design Document
## Version 1.0 | June 2026

---

## 1. OVERVIEW

VoteReederXP is a civic engagement game built into Karen Reeder's campaign website for Texas House District 29. Players earn XP by taking real campaign actions — attending events, knocking doors, registering voters, sharing on social media, and learning about the issues. The game is designed so a dedicated volunteer can realistically reach Change Champion by Election Day (November 4, 2026).

**Core Philosophy:** Every XP point = a real action that helps the campaign.

---

## 2. ACCOUNTS & ACCESS

- **No guest access** — must create an account to play
- **Username rules** — no real names, gamertag style only
- **Starting XP** — 0 XP on signup (being on the leaderboard requires earning it)
- **Welcome experience** — Karen's recorded welcome message plays on first login
- **Audio** — Karen's voice on major rank-ups only, welcome sound on login

## 5. REALISTIC PATH TO CHANGE CHAMPION

A dedicated volunteer doing a little of everything over 5 months (June–November 2026):

| Activity | XP Earned |
|----------|-----------|
| All 10 videos + quizzes | 2,500 XP |
| All 10 lesson completion bonus | 500 XP |
| Attend 8 events (mix of types) | 2,800 XP |
| Event attendance bonuses | 800 XP |
| Voter registration verification | 500 XP |
| Help register 5 voters + bonus | 2,500 XP |
| Knock 50 doors + bonuses | 1,250 XP |
| 20 social posts + bonus | 1,300 XP |
| Recruit 5 volunteers + bonuses | 2,300 XP |
| Volunteer form | 200 XP |
| Donation | 200 XP |
| Civic Quest quizzes (x10) | 1,500 XP |
| **TOTAL** | **~16,350 XP** |

⚠️ **NOTE:** At max single-player effort, ~16,350 XP is achievable. This means **Change Champion at 60,000 XP is not achievable solo.** 

**Recommended adjustment:** Lower Change Champion to **15,000 XP** and adjust all ranks proportionally:

| Rank | Revised XP |
|------|-----------|
| Supporter | 0 |
| Organizer | 300 |
| Advocate | 1,000 |
| Builder | 3,500 |
| Catalyst | 8,000 |
| Change Champion | 15,000 |

## 7. GUILDS (Social Only)

Guilds are for community connection and friendly competition — they do NOT affect individual XP.

**The 5 Guilds:**
- ⚡ The Momentum
- 🔨 Vote Builders
- 🚀 Rise Collective
- 🌊 Turnout Titans
- 🌐 Democracy XP

**Guild features:**
- Guild leaderboard (sum of all member XP)
- Guild chat
- Guild challenges (optional group missions)
- Must be Organizer rank (300 XP) to join

---

## 8. CIVIC CHAMPION OF THE MONTH

**Winner:** Highest XP earned in the calendar month (not total — monthly XP only)

**Announced:** First day of each month

**Prize (free, non-bribe):**
- 🎤 Invited to speak at an upcoming campaign event
- 📜 Official digital certificate signed by Karen
- 📹 Personal thank you video from Karen posted on campaign social media
- 🏆 Featured on website Hall of Fame for that month
- 📛 Special "Civic Champion" badge on their profile

**Rules:**
- Must have attended at least 1 event that month to be eligible
- Karen's team selects from top 3 if winner is inactive
